เที่ยวเมืองไทย ไม่ไปไม่รู้

รวบรวมภาพถ่ายสถานที่ท่องเที่ยวอันงดงามจากทั่วประเทศไทย สัมผัสธรรมชาติ วัฒนธรรม และความทรงจำที่น่าประทับใจ

ทะเล / เกาะ

หมู่เกาะสิมิลัน, พังงา

น้ำทะเลใสราวกับกระจก ทรายขาวเนียนละเอียดดั่งแป้ง สวรรค์ของคนรักการดำน้ำชมปะการัง

ภูเขา / ดอย

ดอยอินทนนท์, เชียงใหม่

ยอดเขาที่สูงที่สุดในประเทศไทย จุดชมพระอาทิตย์ขึ้นและทะเลหมอกเคล้าลมหนาวที่งดงาม

วัด / วัฒนธรรม

วัดพระศรีรัตนศาสดาราม, กรุงเทพฯ

วัดพระแก้วมรกรต โบราณสถานศักดิ์สิทธิ์ใจกลางเมืองหลวง โดดเด่นด้วยสถาปัตยกรรมไทยอันวิจิตร

ทะเล / เกาะ

อ่าวมาหยา เกาะพีพีเล, กระบี่

อ่าวสวรรค์รูปพระจันทร์เสี้ยวที่โอบล้อมด้วยหน้าผาหินปูน หนึ่งในแลนด์มาร์กระดับโลก

ภูเขา / ดอย

ภูทับเบิก, เพชรบูรณ์

สัมผัสไอเย็นและทะเลหมอกเหนือแปลงกะหล่ำปลีสีเขียวสุดลูกหูลูกตาบนยอดเขาสูง

วัด / วัฒนธรรม

อุทยานประวัติศาสตร์พระนครศรีอยุธยา

ย้อนอดีตชมความยิ่งใหญ่ของราชธานีเก่า มรดกโลกทางวัฒนธรรมอันล้ำค่าและน่าหลงใหล
